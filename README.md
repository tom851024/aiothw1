# 互動式線性迴歸分析工具

## 專案總覽

本專案旨在建立一個互動式的網頁應用程式，讓使用者能夠深入理解線性迴歸模型的運作原理。使用者可以透過圖形化介面即時調整資料集的參數，並觀察其對模型訓練結果、效能指標及離群值的影響。此專案的核心是從零開始實作一個線性迴歸模型，並將其部署為一個功能完整的 Streamlit 應用程式。

## 主要功能

- **互動式參數調整**：使用者可以透過側邊欄的滑桿，即時調整以下參數：
  - **資料點數量 (Number of data points)**
  - **真實係數 'a' (Coefficient 'a')**
  - **雜訊變異數 (Noise Variance)**
- **從零實作模型**：線性迴歸模型 (`LinearRegression` class) 完全從頭開始建立，使用梯度下降（Gradient Descent）進行最佳化，不依賴現成的機器學習框架模型。
- **即時視覺化**：
  - 使用 `Matplotlib` 繪製散佈圖，顯示資料點分佈與模型學習到的迴歸線。
  - 自動標示出資料集中 5 個最大的離群值（Outliers），並在圖上顯示其在原始資料中的索引編號。
- **模型係數與效能指標**：
  - **模型係數 (Model Coefficients)**：顯示模型學習到的斜率（Coefficient 'a'）與截距（Coefficient 'b'）。
  - **離群值列表 (Top 5 Outliers)**：以表格形式列出前 5 大離群值的詳細資訊，包含原始索引、X/y 值及預測值。
  - **模型評估 (Model Evaluation)**：計算並呈現均方誤差（MSE）與 R 平方（R-squared）兩個關鍵效能指標。

## 技術棧

- **程式語言**: Python
- **Web 框架**: Streamlit
- **數值計算**: NumPy
- **資料處理**: Pandas
- **圖表繪製**: Matplotlib
- **模型評估**: Scikit-learn (僅用於計算 R² 分數)

## 如何運行

1.  **環境設置**：確保您的環境已安裝 Python 3.x。

2.  **安裝相依套件**：
    在專案根目錄下，執行以下指令來安裝所有必要的函式庫。
    ```bash
    pip install -r HW1/requirements.txt
    ```

3.  **啟動應用程式**：
    執行以下指令來啟動 Streamlit 應用程式。
    ```bash
    streamlit run HW1/app.py
    ```
    執行後，您的預設瀏覽器將會自動開啟應用程式頁面。

## 實施情況

本專案的開發流程遵循了使用者在 `prompt.txt` 中定義的需求。首先，我們建立了一個 `LinearRegression` 類別，其中包含 `fit`（訓練）和 `predict`（預測）方法。模型的訓練核心是梯度下降演算法，透過迭代更新權重（斜率）和偏差（截距）來最小化成本函數（均方誤差）。

接著，我們使用 Streamlit 來建構使用者介面，將資料生成、模型訓練、視覺化和結果分析等環節整合在一起。每一次使用者調整參數，應用程式都會重新生成資料、訓練模型並即時更新所有圖表和指標，提供流暢的互動體驗。

在開發過程中，我們逐步加入了離群值偵測、模型係數顯示等進階功能，並根據使用者的回饋調整了 UI 的排版與細節，最終產出一個功能完整且符合需求的分析工具。

## 結果與結論

此專案成功地將一個從零開始建構的線性迴歸模型，封裝成一個具備高度互動性的 Web 應用程式。最終成果不僅達成了 `prompt.txt` 中設定的所有功能需求，也提供了一個直觀、易於操作的平台，讓非技術背景的使用者也能夠探索和學習線性迴歸的核心概念。

透過這個專案，我們驗證了使用 Streamlit 快速開發資料科學應用的可行性與高效性。從一個簡單的想法出發，我們在短時間內完成了一個功能豐富、反應即時且具備教育意義的實用工具。
